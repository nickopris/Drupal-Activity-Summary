# Drupal Activity Summary
AI-generated summaries of selected Drupal projects.

This repository provides an automated summary of activity pulled from GitLab issues over the last 24 hours.

The data is analysed by a large language model with a focus on topics relevant to executive and developer audiences.

Although the prompts are designed to produce clear, persona-specific summaries, errors can still occur. Please validate important details by reviewing the source data files and the linked issues.

Use the sidebar menu to select the analysis you would like to read.
